## How to Generate Icons

### Method 1: Using HTML Generator (Recommended)
1. Open `html/generate-icons.html` in your web browser
2. Click "Generate All Icons" to preview
3. Click "Download All Icons" to download all three sizes
4. Save the downloaded files to this `icons/` folder:
   - `icon16.png` (16x16 pixels)
   - `icon48.png` (48x48 pixels)
   - `icon128.png` (128x128 pixels)

### Method 2: Using SVG Logo
1. Open `icons/logo.svg` in an image editor (e.g., Inkscape, GIMP, Photoshop)
2. Export/resize to the required sizes:
   - 16x16 pixels ‚Üí `icon16.png`
   - 48x48 pixels ‚Üí `icon48.png`
   - 128x128 pixels ‚Üí `icon128.png`
3. Save as PNG format

### Method 3: Online Converter
1. Use an online SVG to PNG converter (e.g., convertio.co, cloudconvert.com)
2. Upload `icons/logo.svg`
3. Convert to PNG at sizes: 16x16, 48x48, 128x128
4. Download and save to this folder
